Face Detection and Enhancement with OpenCV and Streamlit

Live Project URL : Face Detection App

This project implements a Streamlit web application that performs Image Enhancement tasks such as Gray-scaling images, contrast, brightness and blurring effects of images. Other submodule of the application constitutes for implementing basic computer vision tasks like Face & Eyes Detection and cannizing & cartoonizing images.

This Web Application has been Deployed on Streamlit Sharing.
